O que é html?

voltar

HTML(abreviação para a expressão inglesa Hyper Text Markup Languagem, que sigtnifica: "Linguagem de Marcação de Hipertexto").

HTML serve para estruturar além de textos e imagens, todas as linguagens necessárias para viabilizar diversas plataformas online. Como sites, Blogs, e-commercers. É a estutura básica de tudo que vemos na internet atual (2024).

Tim Berners-Lee (físico britânico) criou o HTML original (e outros protocolos associados como o HTTP),numa estação NeXTcube, usando o ambiente de desenvolvimento NeXTSTEP. Na época, a Linguagem não era uma especificação, mas uma coleção de ferramentas para resolver um problema de Tim: a comunicação e isseminação das pesquisas entre ele e o seu grupo de colegas. A sua solução, combinada com a então emergente internet pública(que tornar-se-ia a internet), ganhou atenção mundial.

As primeiras versões do HTML foram defenidas com regras sintáticas flexíveis, o que ajudou aqueles sem familiaridade com a publiação na Web. Através do tempo, a utilização de ferramentas para autoria de HTML aumentou, assim como a tendência em tornar a sintaxe cada vez mais rígida. Apesar disso, por questões históricas (retrocompatibilidade), os navegadores ainda hoje conseguem interpretar páginas web que estão longe de ser um código HTML válido.

tags h1 ate h6

As tags h1, h2, h3, h4, h5 e h6 trás parâmetros de tamanho de caracteres, sendo h1 o maior e o h6 o menor.

Sintaxe h1

h1 Letra grande /h1

Sintaxe h2

h2 pouco menor que o h1 /h2

Sintaxe h3

h3 pouco menor que o h2 /h3

Sintaxe h4

h4 pouco menor que o h3 /h4

Sintaxe h5

h5 pouco menor que o h4 /h5

Sintaxe h6

h6 pouco menor que o h5 /h6

tags p, mark, smal

voltar

P

Usado para iniciar um parágrafo, ajudando numa organização de textos.

Sintaxe

P Assim, o texto ficará melhor organizado e melhor compreendido para o leitor./P

mark

Faz marcações no texto ou em partes do texto, deixando mais visível ao leitor.

Sintaxe

Assim, podemos marcar o texto inteiro. Assim,marcamos uma só palavra.

small

Usado para diminuir o tamanho de palavras, frases, parágrafos ou até mesmo textos inteiros.

Sintaxe

Assim, posso diminuir o tamanho de uma palavra. Assim, posso diminuir o tamanho de duas palavras Assim, posso diminuir o tamanho de toda a sentença.

tags i, u, strong

voltar

i

Usado para deixar qualquer caracter em itálico, isto é, quando a letra fica um pouco deitada.

Sintaxe

Assim, deixará uma i palavra /i em itálico.

Assim, deixará i duas palavras /i em itálico.

i Assim, deixará toda frase em itálico /i

u

Usado para sublinhar palavras ou até mesmo textos.

Sintaxe

uAssim, o texto ficará tudo sublinhado/u

Assim, apenas uma u letra /u ficará sublinhada

strong

Usado para deixar o caracter num tom negrito, forte, escuro.

Sintaxe

strong Assim, o texto ficará todo escuro. /strong

Assim, uma strong palavra /strong ficará escura

tags ol, ul, li, a

voltar

ol

Usado para criar uma lista ordenada(enumerada), mas na sua sintaxe, precisa usar em seguida a tag li, para listar os ítens.

Sintaxe

ol

    li

  1. casa
    2. /li

    li

  2. quarto
    3. /li

    li

  3. cozinha
    4. /li

    li

  4. banheiro
    5. /li

/ol

ul

Usado para criar uma lista não ordenada(sem numeração de ordem), mas na sua sintaxe, precisa usar em seguida a tag li, para listar os ítens.

Sintaxe

ul

    li

  1. casa
    2. /li

    li

  2. quarto
    3. /li

    li

  3. cozinha
    4. /li

    li

  4. banheiro
    5. /li

/ul

li

Usado para introduzir os ítens de listas ordenas ou não. Esta tag funciona sempre em conjunto com as tags ol e ul.

Sintaxe

ul

    li

  1. martelo
    2. /li

    li

  2. serraa
    3. /li

    li

  3. parafuso
    4. /li

    li

  4. chave de fenda
    5. /li

/ul

a

Usado para criar um hiperlink de dentro da própria página ou de uma página para outra. Mas para isso, precisa ver acrescida da expressão href.

Sintaxe

a href="http://dio.me" target="_blank" site da dio a

OBS:
O termo target="_blank" faz com que o link abra numa outra aba, fazendo com que o usuário permaneça na página.

tags hr, sub, blockquote

voltar

hr

Usado para criar uma linha. Isso divide seções da página e trechos.

Sintaxe

Eu posso escrever algumas linhas e parágrafos. Fazendo que meu texto tenha uma ideia. Após terminar essa parte, colocado uma linha abaixo.

hr

Após essa linha, posso continuar o texto, com outro tema quem sabe. Ou até mesmo continuar o desenvolvimento anterior.

sub

Usado para colocar os caracteres abaixo. Uma boa utilidadade, é para notação expressões matemáticas e químicas.

Sintaxe

Na matemática: Log525 -> Log sub 5 /sub25

Na química: H2O -> H sub 2 /sub O

blockquote

Usado para criar um bloco para notas ou citações. Usado em artigos e também para determinar um bloco para exemplos.

Sintaxe

blockquote

O texto ficará dentro desse bloco, onde assim permanecerá destacado. A cor de fundo foi determinada previamente utilizando a tag style="backgound-color".

/blockquote

tags font, del, abbr

voltar

font

Usado para colocar diversar fontes nos caracteres bem como alterar seu tamanho e cores.

Sintaxe

p style="font-family: verdana;"

Usado para colocar diversar fontes nos caracteres, bem como alterar seu tamanho e cores.

/p

p style="font-size:30"

Nesse exemplo estamos mudando o tamanho da fonte para 30,

/p

del

Usado para deletar palavras e partes de um documento, tracejando-o.

Sintaxe

As escolhar podem parecer estranhas algumas vezes. Um dia é ótimo viajar de avião, mas aproveitamos melhor de carro.

As escolhar podem parecer estranhas algumas vezes. Um dia é ótimo viajar de del avião /del, mas aproveitamos melhor de carro.

Sempre soube como escrever corretamente.

Sempre subi como iscrivi corretamente.

Sempre del subi /del como del iscrivi /del corretamente.

abbr

Usado para colocar abreviações. Quando executado ao passar o cursor na palavra, aparecerá a abreviação completa que foi previamente determinada.

Sintaxe

Muitos países devem dinheiro ao FMI em isso complica suas finanças.

Muitos países devem dinheiro ao abbr title="fundo monetário internacional"FMI /abbrem isso complica suas finanças.
